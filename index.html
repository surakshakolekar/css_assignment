  1. CSS stands for Cascading style sheet
  2. CSS describes how HTML elements are to be displayed on the screen,paper, or in the other media
  3. CSS is not necessary CSS saves lot of work. It can control the layout of multiple webpages at all once.
  4. External stylesheets are stored in CSS files.

CSS can be added HTML document in 3 ways:

  • Inline - by using the style attribute in HTML elements.
  • Internal - by using a style element in the head section.
  • External - by using a link element to link external CSS file.
  • The most common way to add CSS, is to keep the styles in external CSS file. However, in this tutorial we will use inline and internal styles, becuase this is easier to demonstrate, and easier for you to try it yourself

When comparing CSS Class VS ID, the differnce is that CSS class applies a style to multiple elements. ID on other hand, applies style to one unique element,ID is also special in that you can use a special URL to link directly to an element and its used by JavaScript.

In CSS, selectors are used to target to a specific element or a range of elementa on a webpage. Once an element has been targted, a style or set of styles can be applied to the element

There is wide range of selectors available. Two of the most commonly used are Class and ID. Both are used to target elements to which style should be applied.
